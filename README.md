# 📝 To-Do List API

A simple RESTful To-Do List API built with **Java 24**, **Spring Boot**, **MySQL**, and **JPA**.  
This project allows users to create, view, update, and delete tasks.

---

## 🚀 Features

- ✅ Create a to-do item
- 📋 Get all to-do items
- 📝 Update a to-do item
- ❌ Delete a to-do item
- 🧩 Uses MySQL database with JPA/Hibernate
- 🌐 RESTful API endpoints
